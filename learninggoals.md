1. De student kan een relationele databasestructuur ontwerpen, normaliseren en SQL-query's schrijven.
2. De student kan een genormaliseerde databasestructuur ontwerpen en toepassen binnen een applicatie.
3. De student kan geavanceerde SQL-query's schrijven en databaseprestaties optimaliseren.
4. De student kan beveiligings- en privacymaatregelen voor databases implementeren.
5. De student kan een geïntegreerd project opzetten welke gebruikmaakt van verschillende database technologieën.
6. De student kan effectief samenwerken in een teamomgeving, waarbij code wordt beoordeeld, feedback wordt gegeven en communicatie plaatsvindt.
7. De student kan gedetailleerde technische documentatie opstellen die de implementatie, configuratie en gebruik van de ontwikkelde systemen beschrijft.
8. De student kan professioneel communiceren met een opdrachtgever, waarbij eisen worden verzameld, voortgang wordt gerapporteerd en feedback wordt verwerkt in het project.
9. De student kan een gestructureerde en overtuigende presentatie geven over het project, waarbij de belangrijkste resultaten en aanbevelingen duidelijk worden gepresenteerd.
